Identificador: CSU003

Nome: Cadastro de Usuário

Requisito: RF-003 

Responsável: João Pedro Faria Cabral

Descrição/Resumo: Este caso contém o cadastro de usuários/formulário de usuários e permite a inclusão, consulta, alteração e exclusão de dados de usuário.

Atores: Clientes, Bibliotecário(s), Usuários e Advogados.

Pré-Condições: O sistema foi iniciado corretamente e o bibliotecário está logado e autenticado.

Pós-Condições: O formulário de usuário é enviado ao banco de dados, o usuário está pronto para requisitar os serviços da biblioteca.

Cenário Principal:
1: O sistema exige a tela de formulário de usuários.
2: O bibliotecário solicita a inclusão de um novo perfil de usuário.
3: O bibliotecário coloca as informações necessárias para cadastrar um novo usuário.
4: O formulário é registrado no banco de dados.

Cenários Alternativos:
1: O sistema não reconhece os dados de entrada e sugere ao utilizador que cadastre um novo usuário.
2a: O bibliotecário solicita a exclusão de um perfil existente de usuário
2b: O bibliotecário solicita a alteração de um perfil existente de usuário
2c: O sistema informa ao utilizador que o cadastro em questão já existe. 
3: O sistema informa ao utilizador que algum campo não foi preenchido de maneira correta.
4a: O sistema perdeu a conexão ou não pode se conectar ao banco de dados e exibe a mensagem de falha de conexão.
4b: O sistema informa que ocorreu um erro ao cadastrar novo formulário.

Cenários de Exceção:
2a: O usuário solicita ao bibliotecário que interrompa ou cancele o processo de cadastro.
2b: O bibliotecário solicita que o usuário altere algum dado que ele digitou devido a uma incompatibilidade ou impossibilidade no processo de cadastro.
4: O sistema não pode se conectar ao banco de dados.
Requisitos Relacionados: RU-005, RU-006, RU-018, RQ-003
